# 📚 Centro de recursos de aprendizaje

Este repositorio es un centro de recursos diseñado para apoyar el aprendizaje de desarrollo web y otras habilidades clave presentes a lo largo del ciclo de vida del desarrollo de software (SDLC).

El contenido está organizado de acuerdo con las sesiones de estudio, facilitando el acceso progresivo a documentación, videos, artículos y otros materiales de apoyo utilizados en cada tema.

⚠️ Este espacio se encuentra en construcción y seguirá creciendo con el tiempo.

## 🤝 Cómo contribuir

Puedes aportar nuevos recursos, mejoras o correcciones mediante commits o pull requests. Todas las contribuciones serán revisadas antes de integrarse al repositorio.
